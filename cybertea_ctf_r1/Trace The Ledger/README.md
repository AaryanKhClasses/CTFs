# Trace The Ledger
`cybertea`, `osint`

## Problem Statement
One of the internet's original diarists has been logging his life for decades. We need you to find a specific post where he did some number-crunching on his family's engagement with a local public institution. This post, from within the last couple of years, contains the annual usage statistics for two consecutive years. What you need is the full date of this specific blog entry.

Flag Format: CTEA_CTF{date_monthname_year}

### Flag
```
CTEA_CTF{19_january_2024}
```
