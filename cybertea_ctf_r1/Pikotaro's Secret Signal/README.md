# Pikotaro's Secret Signal
`cybertea`, `steganography`

## Problem Statement
The world remembers Pikotaro's catchy hit. But he left a message for only the most dedicated investigators to find. Can you uncover his hidden message?

## My Solution
1. Download the `ppap.wav` [audio](ppap.wav)
2. Open it in `sound-visualiser`
3. Go to Layer -> Add Spectrogram.
4. The flag is hidden in the spectrogram.

### Flag
```
CTEA_CTF{gr34t_mu51c_t4st3}
```
