# FANTASY CTF
Tags: `easy`, `general skills`

## Problem Statement:
Play this short game to get familiar with terminal applications and some of the most important rules in scope for picoCTF. Connect to the program with netcat:
```sh
$ nc verbal-sleep.picoctf.net 63159
```

## My Solution:
1. It is just a RPG. Just follow the instructions by pressing [Enter] whenever you can.
2. When prompted the "registration page", make sure to choose the "Register a single, private account" option.
3. When prompted the "Play a Game" option, make sure to choose the "Play the Game" option.
4. You found the flag.

### Flag:
```
picoCTF{m1113n1um_3d1710n_76b680a5}
```
